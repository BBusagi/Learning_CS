契约：RobotState（遥操作状态包 v1）
—— 一个在网络上同步的机器人/无人机状态快照

字段（按此顺序、此布局）：
  uint32   tick        // 第几帧，权威递增，4 字节
  float    pos_x/y/z   // 位置，各 4 字节
  float    quat_x/y/z/w// 朝向（四元数），各 4 字节
  float    vel_x/y/z   // 速度，各 4 字节
  uint8    status      // 位标志：0x01=已连接 0x02=急停 ...，1 字节
  （对齐/padding 到 4 字节边界）

约束：
  - 总大小固定、无动态分配（要能直接打包成字节流上网）
  - 更新频率 60 Hz（每 16.6ms 一帧）
  - 字节序：小端
  - server 持有权威副本，client 收到后用于渲染/预测
