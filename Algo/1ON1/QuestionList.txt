# 1ON1 问题清单整理版

## 使用方式
- 每次 1ON1 只选 3 到 5 个最重要的问题，避免问题过散。
- 提问顺序建议是：先职业方向，再能力准备，最后落到简历和求职策略。
- 每次对话结束后，补记“关键建议 / 具体行动 / 下次追问”三项。

## 一、职业规划
1. 未来 1 年、3 年、5 年应该如何规划，才能逐步靠近目标岗位？
2. 我目前更适合往哪条路线发展：通用 SDE、后端 / 分布式、AI 工程、XR / 图形底层，还是管理路线？
3. 如果希望转型，应该一步到位，还是先进入相邻岗位再逐步切换？
4. 从市场机会、个人背景和成长上限看，我现阶段最值得押注的方向是什么？

## 二、技术准备
1. 为了匹配目标岗位，我应该优先补哪些基础能力？
2. 算法、系统设计、项目深度，这三项在不同类型公司里的权重分别如何？
3. 在 AI 时代，大厂面试与过去相比发生了哪些变化？
4. 算法准备更适合使用哪种语言：C#、Python，还是其他语言？
5. 如果我继续往 XR / AR / MR / 3D 方向发展，核心技术栈应该如何拆解？

## 三、项目与作品集
1. 我是否还需要准备 Unity 或 XR 相关作品集？
2. 过去偏 toB 的项目经验，应该如何包装成外部更容易理解的展示材料？
3. 展示方式更适合 Demo、视频、开源项目，还是可运行的简化版作品？
4. 如果想补项目，什么样的项目最能证明我具备目标岗位所需能力？

## 四、工程师成长
1. 中级工程师和高级工程师的核心差异是什么？
2. senior 之后，继续往专家路线还是管理路线，通常依据什么来判断？
3. 我现在最欠缺的 soft skills 是什么，应该如何刻意训练？
4. 如何把 AI 工具真正融入日常研发、沟通和产出流程？
5. 多语言能力如何转化为更实际的职业优势，例如跨国协作、客户沟通、国际项目或出差机会？

## 五、求职与简历
1. 我当前的项目经历在市场上的竞争力如何？
2. 简历最需要修改的部分是什么？
3. 面试官会如何评价我当前这份简历，哪些地方会形成强项，哪些地方会直接减分？
4. 我应该如何提升曝光量，增加拿到面试的概率？
5. 目前有哪些公司、岗位或方向特别值得关注？

## 六、重点追问方向
- 当下热门的 XR + AI 结合点有哪些？
- 如果要找练手方向，哪些项目既能学到东西，又能写进简历？
- 外籍候选人在后端、数据平台或核心基础设施岗位上的现实门槛在哪里？
- 是否存在适合多语言背景工程师的岗位类型？

## 七、长期检查项
- 英文面试
- 算法
- 系统设计
