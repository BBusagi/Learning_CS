### 左右指针的常用算法
1. 二分查找
``` python
def binarySearch(nums: List[int], target: int) -> int:
    # 一左一右两个指针相向而行
    left, right = 0, len(nums) - 1
    while left <= right:
        mid = (right + left) // 2
        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            left = mid + 1
        elif nums[mid] > target:
            right = mid - 1
    return -1
```

```python
def isPalindrome(s: str) -> bool:
    # 一左一右两个指针相向而行
    left, right = 0, len(s) - 1
    while left < right:
        if s[left] != s[right]:
            return False
        left += 1
        right -= 1
    return True
```

## 滑动窗口算法核心代码模板
+ 滑动窗口算法技巧主要用来解决子数组问题，比如让你寻找符合某个条件的最长/最短子数组。
+ 类型为如何聪明地穷举
+ （穷举）暴力for循环解法，双层for循环，复杂度为n^2
+ （非穷举）滑动窗口的复杂度是n，区别在于**指针不回退**
+ **滑动窗口并不能穷举出所有子串**
+ 辨别的关键问题
  1. **什么时候扩大窗口？**
  2. **什么时候缩小窗口？**
  3. **什么时候更新结果？**
+ 代码框架，包含输出debug位置，默写出来如下框架然后改三个地方就行
``` python
# 滑动算法，伪代码框架
def slideWindow(s: str):
  # 选择适当的窗口容器来记录数据
  window = xxx
  left, right = 0, 0
  while right < len(s):
    # c 为需要推入的元素
    c = s[right]
    window.add(c)
    # 扩大窗口
    right += 1
    # 窗口内数据更新

    # debug位置
    # *********

    # 判断左侧是否需要收缩
    while left < right and window needs shrink:
      # d 为需要移出的字符
      d = s[left]
      window.remove(d)
      # 缩小窗口
      left += 1
      # 窗口内数据更新
```
+ 可变窗口模板 vs 固定窗口模板
```python
# 可变窗口
while r < len(nums):
  扩大窗口
  while window should shrink:
      收缩窗口
# 固定窗口
for i in range(len(nums)):
    用当前窗口做查询
    把 nums[i] 放进去
    如果窗口大小超过限制，删掉最旧元素
```

## 二分搜索算法核心代码模板
+ 复杂度**O(logn)**
+ 思路简单，但是重在细节
### 二分查找框架
+ 需要看懂[左闭右闭]，方便记忆
+ 推荐写[左闭右开]，条件统一
+ 在二分的时候，**需要保持二分后的区间还是和原来的定义区间一致**
+ 模板
```python
def binarySearch(nums: List[int], target: int) -> int:
    left, right = 0, len(nums) - 1

    while ...:
        mid = left + (right - left) // 2
        if nums[mid] == target:
            ...
        elif nums[mid] < target:
            left = ...
        elif nums[mid] > target:
            right = ...
    return ...
```
+ 左闭右开模板
```python
int binary_search(int[] nums, int target) {
    int left = 0, right = nums.length;    // 左闭右开的搜索区间

    while (left < right) {    // 用 < 而不是 <=，因为 left == right 时搜索区间为空
        int mid = left + (right - left) / 2;
        if (nums[mid] == target) {
            return mid;
        } else if (nums[mid] < target) { // 下一轮搜索区间为 [mid+1, right)
            left = mid + 1;
        } else if (nums[mid] > target) { // 下一轮搜索区间为 [left, mid)
            right = mid;
        }
    }
    return -1;
}
```
+ 先不要使用else，理清所有细节再考虑进行简化
+ 计算mid时需要注意溢出，所以一般采用left + (right-left) / 2 来替代(left + right) / 2

### 搜索区间的概念
+ 二分搜索的本质是利用while调整l和r指针，并不断收缩确保目标数值的索引
  + 搜索区间为空的时候，应该立刻停止
  + 搜索无遗漏
+ 三种使用场景：寻找目标数、寻找左侧边界、寻找右侧边界
1. 寻找一个数 → [Leecode] 704. Binary Search
  + 左闭右开
  + 注意点
    + 循环条件while
    + left和right的更新条件
    + 当出现重复元素的时候，无法区分
2. 寻找左侧边界
  + 找左就缩右侧：`if nums[mid] == target: right = mid;`
  + 返回left
  + 不存在时，返回「大于 target 的最小索引」
3. 寻找右侧边界
  + 找右就缩左侧：`if nums[mid] == target: left = mid + 1;`
  + 返回left-1, right无法取到
  + 不存在时，返回「小于 target 的最大索引」
+ 二分思维：通过已知信息尽可能多地收缩（折半）搜索空间，从而增加穷举效率，快速找到目标

+ [Leecode] 34. Find First and Last Position of Element in Sorted Array
  + 左右两次二分查找，分别得到first和last索引
  + 注意边界条件，需要判断和len(nums) 0 等情况
