## 单调队列结构解决滑动窗口问题
+ 单调队列能够维护最终的同时，保持**先进先出的时间顺序**
+ 优先级队列可以维护最值，但是无法保持时间顺序

+ [Leecode] 239. Sliding Window Maximum
  + 滑动算法来构建框架
  + 使用单调队列进行状态维护（直接使用max复杂度较高）
  + 单调队列的基本api，push,pop,max/min
  + 在push的时候，想象为高个子挤压矮个子，并且注意等值的条件
  + 在pop的时候，需要考虑，自身为最值的时候
  + 数值最多只进出一次，所以复杂度为On

### 拓展
+ min方法，O1时间返回窗口最小值 → 维护一个单调递增队列
+ pop如何不接受val直接删除 → 维护一个原有队列
+ 如果是非固定长度窗口，如何返回窗口size → 维护一个原有队列，和窗口进出保持同步

## 单调栈的几种变体及经典习题
### 单调栈模板的变体

``` python
# 计算 nums 中每个元素的下一个更大元素的索引
def nextGreaterElementIndex(nums):
    n = len(nums)
    # 存放答案的数组
    res = [0] * n
    # 栈中存放索引
    stk = []
    # 因为是求 nums[i] 后面的元素，所以倒着往栈里放
    for i in range(n - 1, -1, -1):
        # 删掉 nums[i] 后面较小的元素
        while stk and nums[stk[-1]] <= nums[i]:
            stk.pop()
        # 现在栈顶就是 nums[i] 身后更大元素的索引
        res[i] = -1 if not stk else stk[-1]
        stk.append(i)
    return res
```

+ 下一个更大的元素索引
`while stk and nums[stk[-1]] <= nums[i]:`

+ 下一个更大或相等的元素索引
`while stk and nums[stk[-1]] < nums[i]:`

+ 下一个更小元素的索引
`while stk and nums[stk[-1]] >= nums[i]:`

+ 下一个更小或相等元素的索引
`while stk and nums[stk[-1]] > nums[i]:`

+ 上一个更大元素的索引
`for i in range(n)`
`while stk and nums[stk[-1]] <= nums[i]:`

+ 上一个更大或相等元素的索引
`for i in range(n)`
`while stk and nums[stk[-1]] < nums[i]:`

+ 上一个更小元素的索引
`for i in range(n)`
`while stk and nums[stk[-1]] >= nums[i]:`

+ 上一个更小或相等元素的索引
`for i in range(n)`
`while stk and nums[stk[-1]] > nums[i]:`
